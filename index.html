Cheat Sheet de Python

Package Managers

1 / 16

¿Qué es un Gestor de Paquetes?

Una herramienta que automatiza el proceso de instalar, actualizar, configurar y eliminar librerías de código creadas por terceros.

2 / 16

PyPI: El Almacén Central

Python Package Index (pypi.org) es el repositorio oficial con cientos de miles de paquetes listos para descargar.

3 / 16

pip: El Instalador Estándar

`pip` es la herramienta por defecto de Python para instalar paquetes desde PyPI. Viene incluido con Python.

4 / 16

Comando: pip install

Descarga e instala la última versión de un paquete y sus dependencias.

pip install requests
pip install pandas
5 / 16

Especificar Versiones

Puedes solicitar una versión exacta o mínima de un paquete.

pip install numpy==1.26.4  # Versión exacta
pip install "requests>=2.31" # Versión mínima
6 / 16

Listar y Mostrar: pip list

Comandos para ver qué tienes instalado en tu entorno actual.

pip list # Muestra todos los paquetes instalados
pip show requests # Muestra detalles de un paquete
7 / 16

Desinstalar: pip uninstall

Elimina un paquete de tu entorno.

pip uninstall pandas
8 / 16

El Archivo requirements.txt

Un archivo estándar para listar todas las dependencias de un proyecto y sus versiones específicas.

requests==2.31.0
numpy>=1.26.0
pandas==2.2.0
9 / 16

Usar requirements.txt

Para guardar tus dependencias actuales y para instalarlas en otro lugar.

# Guardar dependencias actuales
pip freeze > requirements.txt

# Instalar desde el archivo
pip install -r requirements.txt
10 / 16

Contexto: Entornos Virtuales

Es vital usar `pip` dentro de entornos virtuales (`venv`) para aislar las dependencias de cada proyecto y evitar conflictos.

11 / 16

Conda: La Alternativa

Es tanto un gestor de paquetes como de entornos. Muy popular en ciencia de datos. Instala binarios precompilados.

conda install numpy
conda install pandas
12 / 16

Conda Channels

Conda usa "canales" (repositorios). `conda-forge` es un canal comunitario muy popular.

conda install -c conda-forge jupyterlab
13 / 16

pip vs. Conda

pip: Solo paquetes de Python. Usa PyPI. Compila si es necesario.
Conda: Paquetes de cualquier lenguaje. Usa sus propios canales. Binarios listos.

14 / 16

Alternativas Modernas

Herramientas como Poetry y Pipenv combinan la gestión de paquetes y entornos virtuales con una experiencia de desarrollador mejorada.

15 / 16

Conclusión

pip con entornos virtuales es el estándar. conda es excelente para ciencia de datos con dependencias complejas.
